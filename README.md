FongMi影视，TVbox，影音壳子，影视仓。

FongMi：1.点击设定。2.点击点播，把获取到的接口地址粘贴进去，再点确定即可。

TVBox：1.点击设置。2.点击配置地址。3.把获取到的接口地址粘贴进去，再点确定即可。

影音壳子：设置-订阅管理-添加-标题填“OwenTV”，地址填复制的json接口-确定后打开开关。

影视仓：点击设置-点击配置地址-输入推送链接-确定。
